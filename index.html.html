    About This Project

    This interactive explorer presents faculty data extracted from University of Waterloo undergraduate calendar PDFs spanning 1964–1994. The data was extracted using OCR (optical character recognition) and natural language processing to identify faculty names, departments, and academic ranks.

    Coverage: 790 unique faculty members across 45 departments, drawn from 983 individual records. Some years have sparser coverage due to OCR difficulty with older print layouts. Years 1963, 1976, and 1988 had illegible faculty sections and are not included.

    Degree information was extracted from the raw OCR text, which typically included faculty credentials in the format "Name, PhD (Institution)." Not all entries include degree information.

    Inspired by Ian Milligan's UW Faculty History Explorer, which covers 1963–1978 using a similar methodology.

    Data extracted via Python (pdfplumber, pandas). Visualization built with Chart.js.
